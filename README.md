To run this project, you need to have node.js installed.

1.Clone the repo
git clone https://github.com/aurelchivu/Ezugi.git

2.Setup React Application
cd frontend
npm install

3.Setup Node Server
cd ..
npm install

4.Put your video files both in frontend/public/videos and backend/videos

5.Start the servers
npm run dev

6.Enjoy!
